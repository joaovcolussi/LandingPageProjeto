Sobre a Disciplina

A disciplina de Fábrica de Projetos no E-Commerce vai ser utilizado as seguintes tecnologias,wireframe através do Figma, HTML, CSS E JAVASCRIPT como linguagens de front-end. Para back-end vamos utilizaremos C#.

Nossa Empresa

Somos a FitShop, uma startup especializada em soluções inovadoras para e-commerce fitness. Nosso foco é desenvolver um site de e-commerce com os produtos da API do Breda's chamado VendaSimples.

Tema do Projeto

Desenvolvimento de uma plataforma de e-commerce integrada com a API do Breda's, utilizando oque já vimos em aula e com oque estamos aprendendo no momento.
