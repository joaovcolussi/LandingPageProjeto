Sobre a Disciplina

A disciplina de Desenvolvimento de E-Commerce aborda os principais conceitos e tecnologias envolvidas na criação de plataformas de comércio eletrônico. Estudo de casos reais, arquitetura de sistemas, segurança de pagamentos e experiência do usuário.

Nossa Empresa

Somos a FitShop, uma startup especializada em soluções inovadoras para e-commerce fitness. Nosso foco é desenvolver plataformas escaláveis com inteligência artificial para personalização de experiência do cliente e otimização de vendas.

Tema do Projeto

Desenvolvimento de uma plataforma de e-commerce integrada com realidade aumentada para visualização de produtos fitness em 3D. O sistema incluirá recomendações personalizadas usando machine learning e integração com principais gateways de pagamento.
